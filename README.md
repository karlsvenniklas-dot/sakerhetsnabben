# SäkerhetSnabben — Heras Fältservice

Fältverktyg för servicetekniker: BL 401 (brandlarm), BL 402 (inbrottslarm),
BL 445 (lås) och BL 449 (dörrautomatik) med QR-skanning, checklistor,
anmärkningar med foto, protokollexport, historik och offline-lagring.

Helt fristående HTML-app — all data sparas lokalt i webbläsaren.
